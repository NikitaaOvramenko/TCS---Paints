# Quickstart: UI Styling Refresh

## Steps

- Add `TCS-Paint/src/assets/theme.css` with color, spacing, and typography tokens.
- Update components to consume tokens and standard component classes.
- Verify responsiveness at ≤480px, ≤768px, ≤1024px, ≥1440px.
- Confirm contrast with a11y tooling; ensure focus styles on interactive elements.
- Capture before/after screenshots for Home, Gallery, About, Form pages.
